# 📌 Palindrome Checker Web App

A simple and interactive **Palindrome Checker** built using **HTML, CSS, and JavaScript**. This web app allows users to enter text and instantly check whether it is a palindrome.

---

## 🚀 Features

* ✅ Clean and modern UI design
* 🔤 Ignores spaces, punctuation, and capitalization
* ⚡ Instant result display
* 💡 Beginner-friendly project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/palindrome-checker.git
```

2. Open the folder:

```
cd palindrome-checker
```

3. Open `index.html` in your browser

---

## 🧠 How It Works

1. User enters text
2. Convert to lowercase
3. Remove special characters
4. Reverse the string
5. Compare both strings
6. Show result

---

## 📸 Example Output

```
Input: madam
Output: ✅ It is a Palindrome!

Input: hello
Output: ❌ Not a Palindrome.
```
